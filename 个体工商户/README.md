# 福建省个体工商户办理资料夹

这个文件夹用于准备福建省个体工商户设立登记材料，目标是后续申请微信支付、支付宝商户能力。

## 文件说明

- `01-福建个体工商户办理清单.md`：按福建官方口径整理的办理入口、材料和顺序。
- `02-个体工商户登记申请书填写草稿.md`：给网上办事大厅/闽政通填写时对照用的草稿。
- `03-委托朋友办理授权书模板.md`：如果人在美国、线上实名或签名过不了，可让国内朋友先拿模板问窗口。
- `04-经营场所使用文件说明.md`：经营场所证明怎么准备、哪些信息要一致。

## 隐私材料放哪里

身份证照片、银行卡截图、房产证、租赁合同、填写后的含隐私版本，建议放到本目录下的 `private/`、`scans/` 或 `filled/` 子目录。

这些目录和常见图片/PDF格式已被本目录的 `.gitignore` 忽略，避免误提交到 Git。

## 官方入口

- 福建省网上办事大厅：https://zwfw.fujian.gov.cn/
- 福建省政府个体工商户设立登记说明：https://www.fujian.gov.cn/nrrh/sscjgj/202309/t20230927_6265712.htm
- 福建省市场监管局全程网办说明：https://scjgj.fj.gov.cn/zt/sszdgg/202309/t20230907_6251255.htm
