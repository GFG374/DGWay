# 个体工商户登记申请书填写草稿

用途：在福建省网上办事大厅、闽政通 APP 或窗口填写《个体工商户登记（备案）申请书》时对照使用。

注意：这不是官方空白表格原件，是为 DGWay API 收款场景准备的填写草稿。最终以福建省网上办事大厅生成的表单为准。

## 基本选择

```text
申请事项：设立登记
主体类型：个体工商户
组成形式：个人经营
经营方式：网络经营 / 线上服务 / 互联网经营（按系统选项选择）
是否家庭经营：否
是否使用名称：是
```

## 经营者信息

```text
经营者姓名：[你的真实姓名]
证件类型：中华人民共和国居民身份证
证件号码：[拍照/提交时填写，不要写进公开文件]
住所：[身份证或户籍/常住地址，按系统要求填写]
联系电话：[国内手机号，建议能接验证码]
电子邮箱：[常用邮箱]
政治面貌：[按实际选择，如群众]
职业状况：[按实际选择，如自由职业/其他]
```

如果最终由国内朋友实际经营和收款，上面应填写朋友本人信息，不要混用你的业务和朋友的主体。

## 名称备选

个体工商户名称通常包含行政区划、字号、行业、组织形式。以下仅作核名备选，最终以系统通过为准。

```text
备选 1：[城市/区县]某某信息技术服务部
备选 2：[城市/区县]某某软件开发工作室
备选 3：[城市/区县]某某网络技术服务部
备选 4：[城市/区县]某某数据服务工作室
```

如果想和 DGWay 有关联，可考虑：

```text
[城市/区县]鼎维信息技术服务部
[城市/区县]鼎维软件开发工作室
[城市/区县]德维网络技术服务部
```

不要把名称写成容易被误解的平台性质，例如 `支付服务中心`、`代充平台`、`接口代理平台`。

## 经营场所

```text
经营场所地址：[福建省][市][区/县][街道/乡镇][详细门牌号]
场所来源：自有 / 租赁 / 无偿使用 / 住宅改经营性用房（按实际）
产权人/出租人：[姓名或单位名称]
使用期限：[起止日期，按材料填写]
```

地址必须和经营场所使用文件一致。若使用住宅地址，先确认当地是否允许住宅登记为经营场所，以及是否需要业主/物业/社区额外证明。

## 经营范围建议

建议优先填这些低风险、和软件服务匹配的范围：

```text
一般项目：软件开发；信息技术咨询服务；网络技术服务；数据处理服务；互联网数据服务；计算机系统服务；技术服务、技术开发、技术咨询、技术交流、技术转让、技术推广。
```

如果系统要求从经营范围规范目录选择，优先搜索并选择：

```text
软件开发
信息技术咨询服务
网络技术服务
数据处理服务
互联网数据服务
计算机系统服务
```

暂时不要主动加入这些词：

```text
支付
清算
代收
代充
账号共享
翻墙
虚拟币
金融
贷款
投资
```

## 资金数额

如果系统要求填写资金数额，可按实际小额经营资金填写，例如：

```text
资金数额：10000 元
```

个体工商户不是公司注册资本制度，具体是否需要填、怎么填，以福建省网上办事大厅页面为准。

## 从业人数

```text
从业人数：1 人
```

如果国内朋友也作为实际员工或家庭经营成员参与，按实际填写。个人经营通常先填 1 人。

## 网络经营信息

如果系统要求填写网络经营场所或网站信息，可先准备：

```text
网站名称：DGWay API
网站域名：[你的域名]
网站首页：[https://...]
充值页面：[https://.../payment 或实际路径]
ICP备案号：[如已有则填写；没有则先留空或按系统要求处理]
客服手机号：[国内手机号]
客服邮箱：[邮箱]
```

申请微信 PC 网站 Native 支付时，域名和备案信息会更重要。办理个体工商户时，是否必须填网站信息以当地系统页面为准。

## 申请人声明

如果系统生成申请人声明，通常按页面提示勾选或签署。确认前检查：

```text
经营者姓名和身份证一致
经营场所地址和证明材料一致
名称没有错别字
经营范围没有高风险词
联系电话能接通
没有把朋友信息和你的信息混填
```

## 待你补充的材料

```text
经营者身份证正反面照片
经营场所使用文件
国内手机号
具体区县
拟用字号
网站域名和备案情况
银行卡信息，后续申请支付商户号时使用
```
