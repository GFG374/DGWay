# 经营场所使用文件说明

福建省个体工商户设立登记材料里需要“经营场所使用相关文件”。不同区县、不同地址类型要求会有差异，下面用于提前准备。

## 常见地址类型

### 自有房屋

准备：

```text
房屋产权证明/不动产权证相关页面
产权人身份证明
经营者与产权人关系说明，如产权人不是经营者本人
```

### 租赁房屋

准备：

```text
租赁合同
出租方身份证明或营业执照
房屋产权证明/不动产权证相关页面
```

租赁合同里建议包含：

```text
出租方姓名/名称
承租方姓名
房屋详细地址
租赁用途
租赁期限
双方签字/盖章
签署日期
```

### 无偿使用

准备：

```text
无偿使用证明
产权人身份证明
房屋产权证明/不动产权证相关页面
```

无偿使用证明可参考：

```text
经营场所无偿使用证明

本人/本单位同意将位于福建省[市][区/县][详细地址]的房屋，无偿提供给[经营者姓名]用于申请办理个体工商户设立登记及开展软件开发、信息技术咨询等经营活动。

使用期限：自    年    月    日起至    年    月    日止。

产权人/提供方签名或盖章：
证件号码/统一社会信用代码：
联系电话：
日期：    年    月    日
```

### 住宅地址

如果使用住宅地址，先问窗口：

```text
1. 该区县是否允许住宅登记个体工商户？
2. 是否需要利害关系业主同意？
3. 是否需要物业、社区或居委会证明？
4. 是否限制经营范围只能为不扰民的线上服务？
```

软件开发、信息技术咨询这类低扰民业务，一般比实体门店更容易说明，但仍以当地窗口口径为准。

## 地址一致性检查

提交前确认：

```text
申请书经营场所地址
产权证明地址
租赁合同/无偿使用证明地址
门牌号
区县和街道
邮寄地址
```

这些信息尽量完全一致，避免因为简称、错字、漏门牌导致退件。

## 不要提交到 Git 的材料

以下材料放到 `private/`、`scans/` 或 `filled/` 目录：

```text
身份证照片
银行卡截图
产权证照片
租赁合同扫描件
无偿使用证明签字版
填写后的申请书
营业执照照片
```

这些通常包含个人隐私，不要提交到代码仓库。
