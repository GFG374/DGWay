# DGWay 账号购买页面设计

## 目标

在用户端新增“账号购买”页面，为学生用户展示站长自营的账号、长效手机号接码和美国静态住宅 IP 服务。页面仅负责展示商品与联系方式，不接入在线下单、支付、库存或自动发货。

## 页面入口

- 用户侧栏新增“账号购买”，放在“充值/订阅”和“我的订单”之间。
- 管理员侧栏的“我的账户”区域同步显示该入口。
- 路由使用 `/account-store`，登录用户可访问。
- 页面标题为“账号购买”，说明为“账号与长期验证码服务”。

## 视觉设计

- 延续现有 DGWay 的白色、青绿色和紧凑后台界面风格。
- 主内容标题为“站长自营”。
- 标题下方展示“纯手搓账号，一号一 IP，稳定。”
- 桌面端三张商品卡并排；中等宽度两列；手机端单列。
- 卡片圆角不超过 8px，使用不同顶部强调色区分商品。
- OpenAI 商品显示本地内嵌的 OpenAI 标志，避免外链图片加载失败。
- 长效手机号显示手机图标；住宅 IP 显示网络图标。
- 支持浅色和深色模式，按钮和文字满足清晰可读要求。

## 商品内容

### OpenAI 成品账号

- 价格：30 元/个。
- 提供 OpenAI 账号登录邮箱。
- 提供验证码接收网址。
- 管理员确认后人工交付。

### 长效手机号接码

- 价格：10 元/30 天。
- 30 天内可重复接收验证码。
- 最多尝试绑定 3 个 OpenAI 账号。
- 明确提示：实际可绑定数量受 OpenAI 风控影响，不保证一定能够绑定 3 个账号。

### 美国静态住宅 IP

- 价格：40 元/30 天。
- 美国静态住宅网络出口。
- 30 天使用周期。
- 文案说明可降低 OpenAI 与其他 AI 服务的封控风险，不承诺完全消除风险。

## 技术范围

- 新增独立 Vue 页面组件。
- 新增用户路由、侧栏入口和中英文翻译。
- 不新增数据库表、后端接口或管理后台商品管理功能。
- 不把本页面商品计入现有支付订单。
- 更新用户说明文档，说明购买入口、商品内容和人工联系流程。
